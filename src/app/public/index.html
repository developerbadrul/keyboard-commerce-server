Keyboard Commerce Server API Documentation

API Overview

Endpoint Description Method
/api/products Get all products GET
/api/products Add new product POST
/api/products/:id Update product PUT
/api/products/:id Delete product DELETE
/api/products/?search=query Search and filter products GET
/api/order Get all orders GET
/api/order Add new order POST
/api/order/:id Update order status PUT
